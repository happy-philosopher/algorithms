<h1>Алгоритмы и структуры данных</h1>

Для работы с любым языком программирования требуется знать алгоритмы, 
именно в этой теме мы начинаем их изучение. И начинаем мы с основных 
понятий и примеров.

Что такое алгоритмы? Это последовательность действий, которая всегда 
приводит к определенному результату. Говоря простыми словами, это 
перечень указаний для выполнения конкретной задачи. Наиболее широко 
этот термин применяется в информатике и компьютерных науках, где он 
обозначает инструкции, позволяющие эффективно решить задачу.

<h2 id="tema_1">Тема 1. Введение. Линейный поиск и сложности алгоритмов.</h2>

Задача 1:
- Реализуйте алгоритм линейного поиска на языке Python. Функция должна принимать список и
элемент для поиска, и возвращать индекс элемента или -1, если элемент не найден.

Задача 2:
- Проанализируйте временную и пространственную сложность алгоритма линейного поиска.
- Объясните ваши выводы.

Задача 3:
- Создайте список из 100 случайных чисел и выполните линейный поиск для нескольких различных значений.
- Запишите результаты.

Задача 4:
- Сравните время выполнения линейного поиска для разных размеров списков (например,
10, 100, 1000 элементов).
- Постройте график зависимости времени выполнения от размера списка.


<h2 id="tema_2">Тема 2. Массивы и работа с ними. Бинарный поиск.</h2>

Задача 1:
- Реализуйте алгоритм бинарного поиска на языке Python. Функция должна принимать отсортированный список
и элемент для поиска, и возвращать индекс элемента или -1, если элемент не найден.

Задача 2:
- Проанализируйте временную и пространственную сложность алгоритма бинарного поиска.
- Объясните выводы.

Задача 3:
- Создайте отсортированный список из 100 случайных чисел и выполните бинарный поиск для 
нескольких различных значений.
- Запишите результаты.

Задача 4:
- Сравните время выполнения бинарного поиска и линейного поиска на одинаковых списках.
- Постройте график зависимости времени выполнения от размера списка для обоих алгоритмов.


<h2 id="tema_3">Тема 3. Простые алгоритмы сортировки.</h2>

Задача 1:
- Реализуйте алгоритм сортировки пузырьком на языке Python. Функция должна принимать список и возвращать 
отсортированный список.

Задача 2:
- Реализуйте алгоритм сортировки выбором на языке Python. Функция должна принимать список и возвращать 
отсортированный список.

Задача 3. Сравнение алгоритмов:
- Сравните время выполнения трёх алгоритмов сортировки (пузырьком, выбором и вставками) на
одинаковых списках.
- Постройте график зависимости времени выполнения от размера списка для каждого алгоритма.


<h2 id="tema_4">Тема 4. Рекурсия и стек.</h2>

Задача 1:
- Напишите рекурсивную функцию для вычисления факториала числа n.

Задача 2:
- Напишите рекурсивную функцию для вычисления суммы всех элементов в списке.
- Убедитесь, что у функции есть базовый случай.

Задача 3:
- Напишите рекурсивную функцию для выполнения бинарного поиска элемента в отсортированном списке.
Функция должна возвращать индекс найденного элемента или -1, если элемент не найден.

Задача 4:
- Напишите класс Stack, который реализует основные операции стека: push, pop, is_empty и peek.


<h2 id="tema_5">Тема 5. Стек вызовов. Быстрая сортировка.</h2>

Задача 1:
- Напишите рекурсивную функцию для вычисления n-го числа Фибоначчи и проанализируйте стек вызовов для n = 5.

Задача 2:
- Напишите рекурсивную функцию для нахождения максимального элемента в списке, используя подход "Разделяй и властвуй".

Задача 3:
- Напишите функцию для реализации быстрой сортировки (QuickSort) с примером использования.

Задача 4:
- Проанализируйте время выполнения быстрой сортировки на списках различной длины (например, 10, 100, 1000 элементов) 
и сравните её с другими сортировками, такими как сортировка вставками.


<h2 id="tema_6">Тема 6. Очереди и сортировка слиянием.</h2>

Задача 1:
- Напишите класс Queue, который реализует основные операции очереди: enqueue (добавление элемента), dequeue (удаление
элемента), is_empty (проверка, пуста ли очередь) и peek (просмотр первого элемента в очереди).

Задача 2:
- Смоделируйте простой процесс обработки задач в очереди. Допустим, у вас есть список задач, каждая из которых занимает
определенное время на выполнение.
- Используйте очередь для обработки задач и выводите время, когда каждая задача будет завершена.

Задача 3:
- Напишите функцию для реализации сортировки слиянием (MergeSort) с примером использования.

Задача 4:
- Проанализируйте время выполнения сортировки слиянием на списках различной длины (например, 10, 100, 1000 элементов)
и сравните её с другими сортировками, такими как сортировка пузырьком.


<h2 id="tema_7">Тема 7. Хеш.</h2>

Задача 1:
- Напишите класс HashTable, который реализует основные операции хеш-таблицы:
insert (вставка элемента), search (поиск элемента) и delete (удаление элемента).

Задача 2:
- Добавьте в класс HashTable метод resize, который увеличивает размер хеш-таблицы вдвое и
перераспределяет все элементы.
- Протестируйте работу метода на примере хеш-таблицы с начальным размером 5, добавив в неё 10 элементов.

Задача 3:
- Напишите функцию, которая принимает строку и возвращает её хеш-значение.
Для этого используйте простой алгоритм: сложение ASCII-кодов всех символов строки.

Задача 4:
- Используйте написанную хеш-функцию для создания словаря, в котором ключами являются строки,
а значениями — их хеш-значения.
- Реализуйте функции добавления элемента в словарь и поиска значения по ключу.


<h2 id="tema_8">Тема 8. Деревья.</h2>

Задача 1:
- Напишите класс BinaryTree, который реализует основные операции бинарного дерева: вставка (insert) и поиск (search).

Задача 2:
- Напишите функцию для обхода бинарного дерева в ширину (Breadth-First Search, BFS).

Задача 3:
- Напишите три функции для обхода бинарного дерева в глубину (Depth-First Search, DFS):
прямой обход (preorder), симметричный обход (inorder) и обратный обход (postorder).

Задача 4:
- Напишите класс AVLTree, который наследует класс BinaryTree и добавляет операции балансировки
при вставке элементов.
- Реализуйте методы для поворотов (left_rotate, right_rotate) и балансировки (rebalance).


<h2 id="tema_9">Тема 9. Графы.</h2>

Задача 1:
- Напишите класс DirectedGraph, который реализует ориентированный граф с методами добавления вершины
(add_vertex) и добавления ребра (add_edge).
- Протестируйте реализацию на примере.

Задача 2:
- Напишите функцию для обхода ориентированного графа в ширину (Breadth-First Search, BFS).

Задача 3:
- Напишите функцию для создания матрицы смежности из списка ребер.
- Реализуйте функции для добавления вершины и ребра в граф, представленный в виде матрицы смежности.

Задача 4:
- Напишите функции для создания списка смежности из списка ребер.
- Реализуйте функции для добавления вершины и ребра в граф, представленный в виде списка смежности.


<h2 id="tema_10">Тема 10. Динамическое программирование.</h2>

Задача 1:
- Реализуйте функцию, решающую задачу о рюкзаке с помощью динамического программирования. Вам дан рюкзак с
определенной вместимостью и набор предметов с заданными весами и стоимостями. Необходимо найти максимальную
стоимость предметов, которые можно поместить в рюкзак.

Задача 2:
- Реализуйте функцию, которая находит длину наибольшей общей подпоследовательности (LCS) двух строк.

Задача 3:
- Реализуйте функцию, которая находит количество способов разбить число n на сумму чисел, используя динамическое
программирование.

Задача 4:
- Реализуйте алгоритм Флойда-Уоршелла для нахождения кратчайших путей между всеми парами вершин в графе,
представленного в виде матрицы смежности.
